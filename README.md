# Islam Quiz NL

Nederlandse islamitische quiz-app in Kotlin + Jetpack Compose.

## Status
Dit is een eerste testbasis met 100 lokale vragen. Controleer de islamitische inhoud inhoudelijk voordat je publiceert.

## Openen in Android Studio
1. Open Android Studio.
2. Kies **Open**.
3. Selecteer de map `IslamQuizNL`.
4. Wacht tot Gradle sync klaar is.
5. Build/run de app op emulator of telefoon.

## Let op Gradle Wrapper
Als `gradlew`, `gradlew.bat` of `gradle-wrapper.jar` ontbreken, laat Android Studio de Gradle wrapper opnieuw aanmaken of gebruik: `gradle wrapper` op een machine waar Gradle is geïnstalleerd.
